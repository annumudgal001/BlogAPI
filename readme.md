# BlogAPI

A blog API project built using **Node.js**, **Express**, **MongoDB**, and **EJS**. This project was created as a fun and practical exercise to sharpen my web development skills and serves as a professional portfolio piece.

---

## Table of Contents

- [Features](#features)
- [Technologies Used](#technologies-used)
- [Installation](#installation)

---

## Features

- **Admin Authentication:** Secure login and logout using JWT and cookies.
- **Post Management:** Create, edit, and delete blog posts via a protected admin panel.
- **Templating:** Dynamic pages rendered using EJS.
- **Responsive Design:** Clean and modern UI with Font Awesome icons for social links.
- **API-Driven:** Robust API endpoints for all CRUD operations.

---

## Technologies Used

- **Node.js**
- **Express**
- **MongoDB** & **Mongoose**
- **EJS**
- **JWT (jsonwebtoken)**
- **bcrypt**
- **express-session** with **connect-mongo**
- **Font Awesome** (for icons)
